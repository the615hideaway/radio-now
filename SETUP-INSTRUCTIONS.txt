RADIO NOW — SETUP INSTRUCTIONS
===============================

The live site loads song data from a single JSON file:
  data/songs.json

That file is generated from your Google Sheet. The site does NOT read
Google Sheets directly in the browser.


YOUR GOOGLE SHEET
-----------------
https://docs.google.com/spreadsheets/d/1EXNdRluPjwyaY5ktt-qHI2bNF7IT5bD1udnCgkKNdkU/edit

ARTIST SONG SUBMISSIONS (Google Form)
-------------------------------------
Form: https://forms.gle/zFExL6otU1e7hJF59
Embedded on the artist dashboard for reliable file uploads (100 MB per file).

Responses go to spreadsheet tab "Form Responses 1" (you can rename that tab
to "Artist Submissions" — renaming does not break the form link).

Keep one tab per form:
  • Form Responses 1 (or "Artist Submissions") — this song submission form
  • Form Responses DJs — only if you add a separate DJ signup form later
Do not mix song submissions and DJ signups on the same tab.

Google Form theme (to match the Radio Now site):
  • Primary color: #f4c430 (gold)
  • Background: #111 or #1a1a1a (dark)
  • Add section titles: Song details, Vocals & musicians, Release info,
    Press blurb, Audio & cover art, Contact
  • MP3 + WAV: File upload → Audio, max 100 MB
  • Cover Art: File upload → Image, max 100 MB

Required columns (row 1) from the Google Form sheet:
  Artist Name, Song Title, Year, Song Time, Music Style, Description,
  Songwriter, Lead Vocals, Featured Artist, Harmony Vocals 1–4,
  Instrument Player 1–8, MP3, WAV, Cover Art, Website, Record Label,
  Contact E-Mail

Audio previews and downloads use the MP3 column.


SYNC SHEET → JSON (after every sheet update)
---------------------------------------------
Run this in PowerShell from the radio-now folder:

  powershell -ExecutionPolicy Bypass -File scripts\sync-sheet-to-json.ps1

Then commit and push data/songs.json to GitHub to update the live site.

A GitHub Action also syncs the sheet daily at noon UTC (and can be run
manually from the Actions tab).

OPTIONAL — APPS SCRIPT (better audio + ZIP)
-------------------------------------------
For the most reliable in-page audio and ZIP downloads, deploy
google-apps-script/Code.gs to your sheet and paste the Web App URL into
js/config.js as googleScriptUrl. See AUDIO-FIX-STEPS.txt.
